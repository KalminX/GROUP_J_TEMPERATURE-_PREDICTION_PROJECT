## Project Title

**Classroom Temperature Prediction**

## Project Overview

This project applies machine learning techniques to predict the temperature of the 300-level Electrical and Electronics classroom at the University of Benin (UNIBEN). The goal is to model how environmental and occupancy factors influence indoor classroom temperature.

A dataset was collected containing variables such as:

* Humidity
* Number of occupants
* Number of windows open or closed
* Physical dimensions of the classroom (length, width, height)

These features were used to train a machine learning model capable of estimating classroom temperature under similar conditions.

## Workflow

The project workflow consists of:

1. **Data Collection** – Gathering environmental and classroom occupancy data.
2. **Data Preprocessing** – Cleaning, normalizing, and preparing the dataset for model training.
3. **Model Training** – Using Python and the Scikit-learn library to train a predictive model.
4. **Evaluation** – Testing the model’s accuracy in predicting classroom temperature.

The trained model learns relationships between environmental variables and observed temperature values, allowing it to generate predictions when new input data is provided.

## Objective

The project demonstrates how machine learning can be applied to analyze environmental conditions, supporting smarter classroom management and climate monitoring systems.
